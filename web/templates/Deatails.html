How the idea was born?

During our Bachelor's degree in "Azrieli, College of Engineering Jerusalem", at Software Engineering, We used a lot of different links. Every time we sent a link to a friend or group of students, we using the Email or other similar channels.
Over time, we were looking for an easy and convenient way to share links, and succeed to manage the links so we can retrieve them whenever we needed:
therefore we create ShareLink.

What the site offers?

User not registered:

A user can send a link on the 'Main-Page' without any registration to any valid email address.

Registered User:

Personal page - user can save all the links are important to him.
New links page - user can see links received from other users.
If the user wants he can:
share the link in the group, move the link to the personal page or delete link.
Contacts - user can add a contact to his own personal page, and share links.
If the user wants he can delete a contact.
Groups - User can create a group and share links there.
A user can be member of several groups.
If a user has created a group wants to delete her, he can.
